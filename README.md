# Requirements

Python 3.7 or later with the following `pip3 install -U -r requirements.txt` packages:

torch >= 1.1.0
json
unicodedata


# Запуск:

'RunBot.py' запускает бота из консоли:

```bash
python3 RunBot.py
```
# Перед запуском!!!

Распаковываем 'save.zip' в папку Data и готово!
